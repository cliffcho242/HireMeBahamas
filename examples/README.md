# HireMeBahamas Examples

This directory contains example code demonstrating how to use various features of the HireMeBahamas platform.

## Available Examples

### Backend Examples

- `hello_hiremebahamas.py` - A simple "Hello World" style example showing basic usage

### Full Application Examples

- **[my-prisma-postgres-app](./my-prisma-postgres-app/)** - Complete Next.js application using Prisma with Prisma Postgres (Accelerate) for database management
  - Features: Type-safe database access, migrations, seeding, Next.js 15 App Router
  - Perfect for: Learning Prisma setup, building new features with modern database tooling

## Running Examples

### Python Examples

Each Python example can be run directly:

```bash
python examples/hello_hiremebahamas.py
```

### Full Application Examples

See the README in each example directory for detailed setup instructions.

For example, to run the Prisma Postgres app:

```bash
cd examples/my-prisma-postgres-app
# Follow setup instructions in README.md
```

## Contributing Examples

When adding new examples:
1. Use clear, descriptive names
2. Include docstrings/comments explaining what the example demonstrates
3. Keep examples simple and focused on a single feature
4. Include a README.md for full application examples
5. Update this README with the new example
