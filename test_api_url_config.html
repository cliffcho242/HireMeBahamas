Environment Files

For Vite/React apps, create a .env file:

# Local development
VITE_API_URL=http://localhost:8000

# Or for Railway/Render backend
VITE_API_URL=https://your-backend.railway.app

# Or leave empty/unset for Vercel serverless (same-origin)
# VITE_API_URL=

For Next.js apps, use NEXT_PUBLIC_ prefix:

# Next.js
NEXT_PUBLIC_API_URL=https://your-backend.com
